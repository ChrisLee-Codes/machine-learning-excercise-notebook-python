# 机器学习Jupyter Notebook

但其assignment使用了Matlab/Octave语言，这显然已经不适合当前机器学习/深度学习技术的发展。

此Repository将其所有课程assignment内容更新为Python版本，并采用Jupyter Notebook的形式，方便阅读和实验。
